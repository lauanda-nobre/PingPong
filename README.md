<h1 align="center">Ping Pong</h1>

<b>Este projeto é um jogo de Ping Pong controlado por um bot.  </b>
 
> O conceito e o visual são inspirados nos primeiros jogos de Ping Pong desenvolvidos para computador, especialmente aqueles produzidos durante as décadas iniciais da computação gráfica.
> A proposta do projeto é explorar uma abordagem simples e direta, tanto na jogabilidade quanto na implementação técnica, priorizando clareza de código e organização dos componentes do jogo.

#### 1.1 Projeto e Implementação

O jogo foi desenvolvido utilizando o framework **p5.js**, que é voltado para programação criativa e facilita a construção de elementos visuais por meio de código.

Um dos principais motivos para a escolha do p5.js é a possibilidade de criar toda a parte gráfica e lógica do jogo de forma integrada, com uma quantidade reduzida de código JavaScript em comparação a abordagens mais tradicionais. Além disso, o framework permite uma separação clara entre os diferentes componentes do jogo, o que contribui para uma melhor organização do projeto.

A implementação é dividida em múltiplos arquivos JavaScript, onde cada arquivo é responsável por uma parte específica do funcionamento do jogo. Essa divisão torna o código mais legível e facilita futuras modificações ou expansões.

No fluxo final de execução, todos esses arquivos são importados por um arquivo principal, que é responsável por reunir os scripts, inicializar o ambiente gráfico e executar o jogo.
